# Descripciones de puesto · formato Upwork

Una por rol, **en inglés** (el estándar del marketplace) y listas para copiar y pegar al
publicar. Cada una cumple los requisitos de una buena publicación en Upwork:

1. Título claro y específico
2. Descripción del proyecto (overview)
3. Responsabilidades y entregables concretos (por milestone/gate)
4. Habilidades y calificaciones requeridas (+ deseables)
5. Nivel de experiencia (Expert / Intermediate)
6. Duración del proyecto y horas semanales
7. Tipo de presupuesto (precio fijo por milestone — **los montos se capturan al publicar**,
   nunca en este repo público)
8. Preguntas de filtrado (screening questions)

**Al publicar:** usar contrato de precio fijo con un milestone por gate (los entregables ya
están listados en cada JD); pedir portafolio en las screening questions; activar
"Featured/Invite-only" para los roles Expert si el flujo de candidatos es bajo.
